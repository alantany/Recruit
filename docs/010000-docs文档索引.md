# docs 文档索引（编号说明）

## 编号规则

- 文件名前缀 **`01xxxx`**：`01` 固定段 + **四位序号**（`0000`～`9999`）。
- **`010000`**：本索引与项目导读。
- **`010001` 起**：正文文档，**按文件在仓库中的最后修改时间（mtime）从早到晚** 编号；同一天修改的，按文件名字符顺序细分。

## 项目概况（读文档后的摘要）

- **目标**：智联招聘（`rd6.zhaopin.com`）上，用 Playwright 自动化完成搜索/推荐/潜在人选触达、互动区回复、简历下载与状态落库，减轻重复劳动。
- **主线 A（当前主推）**：`search-interaction:loop` — 同一浏览器会话内「搜索人才 ↔ 互动区」循环；配置见 `config/recruit-agent.json`。
- **主线 B**：推荐/潜在 → 评分 → 触达 → 互动；另有 `workflow:run`、`daemon:run` 等编排。
- **代码**：`src/recruit-agent`；技能说明：`skills/recruiting-zhilian/SKILL.md`。
